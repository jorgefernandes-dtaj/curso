Principais formatações

Negrito / Destaque

Usar a tag b para escrever em negrito (obsuleto)

usar a tag strong para dar destaque

Italico / Enfase

usar a tag i para escrever em italico(obsuleto)

usar a tag em para dar enfase

Texto marcado

Podemos criar um texto marcado

Pdemos mudar a cor do tag individualmente ou geral usando css

Texto grande e pequeno

Criar um texto grande com big (obsuleto) e um texto pequeno com small

Texto deletado

Podemos marcar um texto como excluido com del

Texto inserido

Podemos marcar um texto inserido com ins ou u (obsuleto)

Texto sobrescrito

Para inserir coisas do tipo x20+3

Texto subscrito
